## 🔧 트러블슈팅 가이드

### 자주 발생하는 문제

```markdown
### Flutter 빌드 오류
증상: flutter build 실패
해결: flutter clean && flutter pub get && flutter build

### iOS 코드 서명 오류
증상: Xcode 빌드 시 signing 오류
해결: Xcode에서 Signing & Capabilities 확인, 
      Apple Developer 인증서 갱신

### Android Health Connect 권한 오류
증상: 권한 요청 후에도 데이터 접근 불가
해결: AndroidManifest.xml 권한 확인,
      테스트 기기에 Health Connect 앱 설치 확인

### OpenAI API 오류
증상: 429 Too Many Requests
해결: API 호출 빈도 조절, 재시도 로직 추가

### WebSocket 연결 끊김
증상: 상담 중 연결 끊김
해결: 자동 재연결 로직 확인, 네트워크 상태 체크
```
