## 1. 설계 철학 및 원칙

### 1.1 5가지 설계 기둥 (업데이트)

```
1️⃣ 신뢰 (Trust)
   의료 정보와 제품 추천의 정확성
   → 식약처 인증 마크, 성분 투명 공개
   
2️⃣ 친근함 (Warmth)
   가족의 건강을 함께 돌보는 따뜻함
   → 부드러운 톤, 격려 메시지
   
3️⃣ 간결함 (Simplicity)
   복잡한 건강정보와 제품 정보를 쉽게
   → 카드 형태, 시각적 요약
   
4️⃣ 지속성 (Retention)
   매일 사용하고 싶은 경험
   → 스트릭, 성취감, 제품 혜택
   
5️⃣ 🆕 투명성 (Transparency)
   제품 추천의 근거와 한계 명확히
   → 면책 조항, 금기 사항 강조
```

### 1.2 제품 추천 디자인 원칙 🆕

```
🎯 자연스러운 통합
   상담 맥락에 맞게 추천
   → 강요하지 않음
   → 사용자가 선택적으로 확인

✅ 신뢰 구축
   식약처 인증 마크 강조
   → 성분 정보 상세 표시
   → 의료진 검수 표시

⚠️ 안전 우선
   금기 사항 명확히 표시
   → 개인 건강 상태 고려
   → 면책 조항 항상 표시

💚 개인화
   "엄마에게 적합" 메시지
   → 나이, 질환 고려 표시
   → 맞춤형 주의사항
```

## 13. 사용자 여정 맵

### 13.1 제품 추천 여정 🆕

```
1️⃣ 증상 호소
   사용자: "요즘 눈이 피로해요"
   ↓
2️⃣ AI 조언 + 추천 제안
   AI: "루테인이 도움될 수 있어요. 제품 추천해드릴까요?"
   ↓
3️⃣ 제품 카드 확인
   사용자가 3개 제품 캐러셀 확인
   ↓
4️⃣ 제품 상세 보기 (선택)
   성분, 인증, 개인화 정보 확인
   ↓
5️⃣ 구매 결정
   Phase 1: 외부 쇼핑몰 링크 클릭
   Phase 2: 장바구니 담기 → 결제
   ↓
6️⃣ 구매 완료 (Phase 2)
   주문 추적, 리뷰 작성, 포인트 적립
```

### 13.2 정기 배송 여정 (Phase 2) 🆕

```
1️⃣ 제품 구매 경험
   만족스러운 첫 구매
   ↓
2️⃣ 재구매 필요성 인지
   "한 달 후 다시 사야겠네"
   ↓
3️⃣ 정기 배송 제안
   결제 화면: "정기 배송으로 10% 할인"
   ↓
4️⃣ 정기 배송 신청
   주기 선택 (1/2/3개월)
   ↓
5️⃣ 자동 배송
   알림: "3일 후 정기 배송됩니다"
   ↓
6️⃣ 관리
   언제든 변경/일시정지/취소 가능
```

struct ProductCard: View {
    let product: Product
    let healthContext: HealthContext
    
    var body: some View {
        VStack(alignment: .leading, spacing: 12) {
            // 제품 이미지
            AsyncImage(url: product.imageURL) { image in
                image
                    .resizable()
                    .aspectRatio(contentMode: .fill)
            }
            .frame(width: 150, height: 150)
            .cornerRadius(8)
            
            // 제품명

                .font(.headline)

                .font(.subheadline)

                Image(systemName: "checkmark.seal.fill")
                    .foregroundColor(.green)
                Text("식약처 기능성 인증")
                    .font(.caption)
            }
            
            // 성분 정보
            VStack(alignment: .leading) {
                ForEach(product.keyIngredients) { ingredient in
                    HStack {
                        Text("•")
                        Text("\(ingredient.name) \(ingredient.amount)")
                    }
                    .font(.caption)
                }
            }
            
            // 개인화 메시지
            if product.isSafeFor(healthContext) {
                HStack {
                    Image(systemName: "heart.fill")
                        .foregroundColor(.green)
                    Text("\(healthContext.profileName)님께 적합")
                        .font(.caption)
                }
                .padding(8)
                .background(Color.green.opacity(0.1))
                .cornerRadius(8)
            }
            
            // 가격
            HStack {
                Text("₩\(product.discountedPrice)")
                    .font(.title3)
                    .bold()
                
                if product.discount > 0 {
                    Text("\(product.discount)% ↓")
                        .font(.caption)
                        .foregroundColor(.red)
                }
            }
            
            // 구매 버튼
            ForEach(product.purchaseLinks) { link in
                Button(action: {
                    openURL(link.url)
                }) {
                    HStack {
                        Text("\(link.platform)에서 보기")

                    }
                    .frame(maxWidth: .infinity)
                    .padding()
                    .background(Color.blue)
                    .foregroundColor(.white)
                    .cornerRadius(8)
                }
            }
        }
        .padding()
        .background(Color(.systemBackground))
        .cornerRadius(12)
        .shadow(radius: 4)
    }
}
