## 🎯 핵심 설계 철학

### "신뢰 + 친근함"

```
기존 의료앱의 문제:
❌ 신뢰만 → 차갑고 딱딱함
❌ 친근함만 → 낮은 신뢰도

우리의 접근:
✅ 신뢰 + 친근함 동시 달성
   → 편하면서도 안심
```

### 5가지 설계 기둥

1. **신뢰** (Trust): 의료 정보 정확성, 식약처 인증
2. **친근함** (Warmth): 가족 중심, 공감하는 언어
3. **간결함** (Simplicity): 3클릭 내 목표 달성
4. **지속성** (Retention): 습관 형성, 스트릭 시스템
5. **투명성** (Transparency): 데이터 사용 투명 공개

## 📱 주요 화면 구조

### 1. 홈 화면
```
헤더 (인사, 알림)
↓
가족 프로필 선택 (스크롤)
↓
오늘의 건강 요약 (카드)
↓
오늘의 루틴 (3/5) (카드)
↓
최근 상담 내역 (카드)
↓
건강 팁 (카드)
↓
하단 네비게이션
+ FAB (우하단)
```

### 2. 음성 상담
```
대상 선택
↓
AI 캐릭터 선택
↓
음성 상담 (실시간 파형)
↓
상담 요약
↓
제품 추천 (선택적)
```

### 3. 가족 프로필
```
프로필 목록
↓
프로필 추가/수정
   ├─ 기본 정보
   ├─ 건강 정보
   └─ 웨어러블 연동
```
